# Dashboard de Gestión de Tareas

Interfaz web moderna para la gestión de tareas personales, desarrollada con React y TailwindCSS. Este proyecto consume la API REST de Tareas.

## Características

- Autenticación de usuarios (Registro e Inicio de sesión).
- Gestión de tareas (Crear, Leer, Actualizar, Eliminar).
- Diseño responsivo y amigable.
- Rutas protegidas y manejo de sesiones con JWT.

## Requisitos Previos

- Node.js (Versión LTS recomendada).
- API de Tareas en ejecución (puerto 8000).

## Instalación y Ejecución

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Iniciar servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Abrir en el navegador:
   http://localhost:5173

## Tecnologías

- React
- Vite
- TailwindCSS
- React Router DOM
